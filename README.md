A Restaurant Management System developed using Laravel to simplify restaurant operations. The system allows customers to browse menus, book tables, place orders, and submit feedback, while administrators can efficiently manage meals, categories, orders, bookings, and users.

---

## 📌 Features

### 👤 User
- User Registration & Login
- Browse Restaurant Menu
- View Meal Details
- Book a Table
- Place Orders
- View Order History
- Submit Feedback
- Contact Restaurant

### 👨‍💼 Admin
- Secure Admin Login
- Dashboard
- Manage Categories
- Manage Meals
- Manage Table Bookings
- Manage Orders
- Manage Users
- View Customer Feedback

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/jiyaa2711/restaurant-management-system.git
```

2. Go to project folder

```bash
cd restaurant-management-system
```

3. Install dependencies

```bash
composer install
```

4. Copy environment file

```bash
cp .env.example .env
```

5. Generate application key

```bash
php artisan key:generate
```

6. Configure database in `.env`

7. Run migrations

```bash
php artisan migrate
```

8. Start the server

```bash
php artisan serve
```
